
## 專案介紹：

   ### 1. 數據來源：
          1. 本專案是利用爬蟲抓取ＧＯＯＧＬＥ評論進行深度學習的模型訓練。
          2. 爬蟲技術使用Requests、Selenium及PyAutoGUI 進行。
          3. PyAutoGUI 可以開啟開發者工具，抓取爬蟲需要的 JSON 網址，最後丟到Requests進行數據大量抓取。
